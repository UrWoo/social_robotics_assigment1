# Social robotics assigment 1

Project that enables user to play a game With other words with an AlphaMini robot.

## Installation

Download the repository and install Python (>=3.10). You also need your own API key for gemini, which should be saved in an enviroment variable as `GEMINI_API_KEY` or `GOOGLE_API_KEY`. To do this follow the guide at https://ai.google.dev/gemini-api/docs/api-key

In the project folder run the following script to install dependencies:
```bash
pip install -r requirements.txt
```
Then in the code change the variable `realm` to your generated realm.

Then simply run the code using 
```bash
python3 main.py
```
